Android-Performance-Tool--Eclipse-plugin- by gustarim

The Android Performance Tool is an Eclipse plugin that allow the developer of an Android application to automatically check the performance guidelines given in the Android Dev Guide (http://developer.android.com/guide/practices/design/performance.html)

In the Google Android developers web site (http://developer.android.com) in the Dev Guide it is possible to find a section that describes the Best Practice to follow in order to write an android application. One of its subsections gives a detailed list on how to design applications for performance (http://developer.android.com/guide/practices/design/performance.html). Here you can find a series of guidelines and some of them can be checked automatically by looking the byte code of the .class Java files. Unfortunately Google doesnâ€™t provide a tool that can check automatically those guidelines and the developer must remember and apply all the suggestions on its own. The aim of this project is to provide the missing tool to the developers of the Android applications in order to allow a developer to automatically check all the performance guidelines.
